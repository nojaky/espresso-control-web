# 에스프레소 컨트롤 차트 Web

파이썬 Tkinter 버전의 `에스프레소 컨트롤 차트`를 GitHub Pages에서 실행할 수 있는 정적 웹앱으로 변환한 버전입니다.

## 구성

- `index.html`: 화면 구조
- `styles.css`: 반응형 UI 스타일
- `app.js`: 계산 로직, 에스프레소 차트, 측정 기록, CSV 저장

## 변환된 기능

- 추출된 커피 용량, 사용한 원두 용량, TDS 입력
- 추출 수율 계산
- 에스프레소 컨트롤 차트 표시
- SCAA 추출수율 18~22%, TDS 6.5~10.5% 영역 표시
- 리스트레또, 에스프레소, 룽고 및 과소/과다 추출 판정
- 측정 기록 저장, 기록 재입력, 기록 삭제
- 측정 기록 CSV 저장
- 네이버 톡톡 문의 버튼

## 배포

GitHub 저장소에 파일을 업로드한 뒤 Pages를 켜면 바로 사용할 수 있습니다.
